# Project DNA

Project DNA is an AI Architecture Governance Platform designed to preserve the architectural DNA of software projects and reduce architectural hallucination in AI-assisted development.

## Features

- TypeScript-based CLI foundation
- Layered architecture for future extensibility
- Placeholder commands for init, ask, and validate
- Extensible JSON memory files for project architecture context

## Commands

- `pdna init`
- `pdna ask`
- `pdna validate`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
